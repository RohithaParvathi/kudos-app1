# Kudos App

## Backend (Django)
1. `cd backend`
2. `python -m venv env && source env/bin/activate`
3. `pip install -r requirements.txt`
4. `python manage.py migrate`
5. `python manage.py generate_demo_data`
6. `python manage.py runserver`

## Frontend (React)
1. `cd frontend/kudos-ui`
2. `npm install`
3. `npm run dev`
